Building a diversified crypto portfolio usually means researching multiple coins, deciding how much to allocate to each, and placing several separate trades. Auto Buy Bundle simplifies that process into a single click.

### **1. What is Auto Buy Bundle?**

Auto Buy Bundle lets you invest in a pre-set selection of cryptocurrencies in one transaction, with your funds automatically split across each coin in the bundle. Instead of buying each asset individually, you can gain exposure to multiple coins at once — with the allocation already worked out for you.

CoinByte currently offers four bundles, each built around a different level of diversification.

Each bundle also displays a 5-year return figure in the app, giving you a quick reference for how that combination of coins has historically performed. Past performance doesn't guarantee future results, but it can be a useful starting point when comparing bundles.


### **3. How It Works**

    3.1 Open Crypto Bundles from the Auto Buy page.
    3.2 Choose a bundle based on how diversified you'd like to be — from 2 coins up to 10.
    3.3 Review the allocation for each coin in the bundle and set Auto-buy order amount.
    3.4 Confirm your purchase.
