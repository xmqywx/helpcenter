We send SMS codes to the phone number linked to your CoinByte account. Please note that:

<TickList>
  <li>Codes may take a few minutes to arrive</li>
  <li>Each code is valid for 10 minutes</li>
  <li>Your account will be locked if you enter the incorrect code 6 times within a short time frame</li>
</TickList>

---

If you're not receiving the codes, try the following solutions:

<TickList>
  <li>Clear your browser or app cache and then request a new code</li>
  <li>Try requesting the code from different CoinByte platforms: desktop browser, mobile browser, or mobile app</li>
</TickList>

---

If you're still unable to verify your mobile number, consider using alternative methods. Follow the steps below to find out how:

## 1. Choose Google Verification as an alternative

About how to set up Google Verification, please refer to <a href="/helpcenter/docs/account-security/verification-and-security/set-up-two-factor-authentication"><strong>Set up two-factor authentication</strong></a>.

## 2. Change another phone number

About how to change another phone number please refer to <a href="/helpcenter/docs/account-security/update-details/change-your-phone-number"><strong>Change your phone number</strong></a>.

---

If the above solutions still do not resolve the issue, please contact our customer support team by emailing <a href="mailto:cs@coinbyte.com.au">cs@coinbyte.com.au</a> or via the online chatbot. Please use your email linked to the CoinByte account with login issue to contact us.
